# A/B Testing: Subscription Conversion Optimization

## 📊 Description
This project analyzes the impact of different pricing messages on user conversion to a paid subscription.

## 🛠 Tools
- Python (Pandas, Matplotlib, SciPy)
- Google Sheets

## 🔍 What was done
- Prepared and analyzed A/B test data
- Calculated conversion rate and uplift
- Performed statistical significance testing (t-test)
- Visualized results (conversion rate, trends over time)
- Created presentation with conclusions

## 📈 Result
Variant B showed higher conversion compared to Variant A.

The result is statistically significant (p-value < 0.05), therefore Variant B is recommended for implementation.
